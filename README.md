# OpenCart Manual Testing Project

## 📌 Project Overview
This project contains manual testing documentation for the OpenCart ecommerce website. The testing was performed to validate important functionalities such as user authentication, product search, cart management, and checkout workflow.

---

## 🌐 Website Tested
https://demo.opencart.com/

---

## 🎯 Testing Objectives
- Verify major ecommerce functionalities
- Identify functional and UI defects
- Validate cart and checkout processes
- Ensure proper user interaction and navigation

---

## 🧪 Testing Types Performed
- Functional Testing
- UI Testing
- Regression Testing
- Negative Testing

---

## 📋 Modules Tested
- User Registration
- Login & Logout
- Product Search
- Product Details
- Add to Cart
- Shopping Cart
- Checkout Process
- Navigation Menu

---

This repository contains:
